# Loco API Documentation

API documentation for Loco API built with [Redocly](https://redocly.com/).

## Setup

```bash
git clone https://github.com/beacrewjp/loco-api-document.git
cd loco-api-document
npm install
```

## Commands

```bash
npm run lint      # Validate OpenAPI spec
npm run preview   # Start local preview server
npm run build     # Generate static HTML
```
